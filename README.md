# Tutor Management SQL Database Project

#Overview
This project is a relational database built using SQL.  
It models tutors, subjects, and teaching sessions to demonstrate database design and query skills.

#Database Structure
The database includes three related tables:
-tutors (stores tutor information)
-fields (stores subjects taught by tutors)
-hours(tracke teaching sessiong and dates)

#skills Demonstrated
-SQL table creation (DDL)
-Primary and foreign keys
-Table relationships
-JOIN operations
-Data filtering and aggregation (GROUP BY, SUM)

#How to Run
1. Run schema.sql first to create tables
2. Insert sample data
3. Run queries.sql to analyze data

#Notes
This project uses fictional data for learning purposes and SQL practices.
